Portfolio

  • Reversi

    The difficulty in programming Reversi is the dynamic nature of the game play. A move could take anywhere between one and six opponent pieces in a given move.


    Key Function: searchBoard compiles the surrounding squares' values in a given direction into a string, making it easy for a regex to find a valid move.

    Techniques Used:

    Recursive Function Calls
    Complex Logic
    Model/View Separation
    DOM Manipulation with JQuery
    Regex for Move Validation

  • Derry's Fishin' Friends

    I teach at a bootcamp, and for the past year, we've taught Angular. My students needed a reference app for a MEAN app, so I built Derry's Fishin' Friends, a full stack example complete with authentication, persisted data, and AJAX.


    Key Function: setUser takes the JSON Web Token off of Local Storage, decodes it using the browser's built in atob decoder and sets the user on the UserDataService for use across all controllers.

    Techniques Used:

    MEAN Stack
    JWT Authentication
    ui-Router
    AJAX using $http
    Local Storage
